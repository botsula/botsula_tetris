# jstetris-learn

1) `npm instal`

2) `serve`

## How to play?

To **move** use arrows: ` ← `, ` → `, ` ↓ `.

To **rotate** use ` z `.

To **pause** use ` spacebar 🌌🍷`.

### What is available in current version:

- generate new figure
- move right, left, down
- rotate object (4 angles)
- delete full row
- pause the game
